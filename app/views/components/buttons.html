Buttons buttons.js

Example uses

Do more with buttons. Control button states or create groups of buttons for more components like toolbars.

Single toggle

Add wg-button to activate toggling on a single button.

button: {{button | json}}

Select toggle

Add wg-button-select to activate select-like toggling on a single button.

buttonSelect: {{buttonSelect | json}}

Checkboxes

Add wg-buttons-checkbox to a group of checkboxes for checkbox style toggling on btn-group.

checkbox: {{checkbox | json}}

Radios

Add wg-buttons-radio to a group of radio inputs for radio style toggling on btn-group.

radio: {{radio | json}}
